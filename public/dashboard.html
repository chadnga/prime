⚡ Advanced Auto Moderation

Toggle automatic actions for common spam and abuse patterns

Spam

Channel Spam
Take actions when a user sends messages across multiple channels too fast.
Mentions
Take actions when a user mentions too many users in a short time.
Attachments
Take actions when a user sends too many attachments in a short time.
Emojis

Message Lines

Capitalization

Links
Take actions when a user's message includes links.
Discord Invites

Commands

Configure bot response behaviour for error messages

Error Messages

Control which error responses the bot sends to users

Command not found
Sent when an executed command doesn't exist.
Wrong command usage
Sent when an existing command is used incorrectly.
No command permissions
Sent when an unpermitted user is executing an existing command. If disabled, "Command not found" will be sent instead.
Disabled in channel
Sent when an existing command is executed in a channel where it's disabled.
Command on cooldown
Sent when an existing command is executed while the user is on cooldown.
